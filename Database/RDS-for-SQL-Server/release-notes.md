### 2023. 03. 14.
#### 機能改善

* オブジェクトストレージに全体バックアップする機能を削除

* オブジェクトストレージに差分バックアップする時、現在のデータベース基準で選択できるように機能改善
* オブジェクトストレージにあるバックアップで復元する時、差分バックアップを一緒に復元するように機能改善
* オブジェクトストレージにバックアップをエクスポートする時、バックアップされたデータベースを選択できるように機能改善

#### バグ修正

* オブジェクトストレージにあるバックアップで復元する時、差分バックアップは復元できない現象を修正
